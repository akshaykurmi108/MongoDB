# MongoDB for Beginners

A collection of simple MongoDB programs to help beginners learn the basics of MongoDB — including CRUD operations, queries, and database concepts.

## What's Inside
- Basic Create, Read, Update, Delete (CRUD) operations
- Simple queries and filters
- Beginner-friendly, well-commented code

## Requirements
- MongoDB installed locally (or MongoDB Atlas)
- basic knoof DataBase

## How to Run
1. Clone the repository
2. 2. Install dependencies
3. Run the scripts and see the output

## Purpose
This repo is meant for anyone starting out with MongoDB and wants hands-on, simple examples to practice.
